⚡ Optimizer X



Optimizer X is a powerful, all-in-one PowerShell utility designed to drastically enhance, debloat, and manage Windows 10 and 11 environments. It features a modern, responsive WPF graphical interface and provides deep system optimizations, hardware-specific tweaks, and an integrated undo manager for total safety.





✨ Key Features



Smart Hardware Detection: Automatically adapts tweaks based on your CPU, RAM, GPU, RAID status, and whether you use an SSD or an older HDD.



Bilingual Interface: Fully translates between English and French on the fly.



12+ Optimization Categories: Advanced (Expert), Disk, Gaming, Maintenance, Network, Operating System, Privacy, Restore, Software (Winget), Storage \& NVMe, System Services and Visual Interface.



Built-in Undo Manager: Safely revert any registry and service modifications using the automatically generated "Registry\_Undo.csv" log.



Live System Monitor: Real-time tracking for CPU, RAM, Disk, Network, and GPU usage directly within the dashboard.



One-Click Profiles: Pre-configured optimization profiles for Gaming (Maximum), Office (Stable), and Cleanup (Minimum).





🛠️ Prerequisites



Windows 10 or Windows 11



Administrator Privileges (The script will automatically prompt for elevation if needed)





📥 Installation \& Usage



1. Download the latest version of the script or clone this repository.
2. Right-click "OptimizerX.exe" and select "Run as Administrator".
3. Select your desired language at the top right of the application (EN / FR).
4. Browse the left navigation menu and check the optimizations you want to apply.
5. Click the "APPLY SELECTION" button at the bottom right.
6. Review the generated "Optimization\_Report.txt" in the root folder to see exactly what was executed.





⚠️ Disclaimer



ATTENTION: This tool modifies core Windows settings, registry keys, and system services. While a granular Undo Manager is included, it is highly recommended to create a system restore point before applying major changes. USE AT YOUR OWN RISK!
